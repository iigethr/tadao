# Tadao

A Minimal and Lightweight Design System.

### Usage

#### Tadao on its own
Add this line to your application.css

```
*= require tadao
```

### Installation

Add this line to your application's Gemfile:

```
gem "tadao"
```

Execute:

```
$ bundle install
```

Or install it yourself as:

```
$ gem install tadao
```
